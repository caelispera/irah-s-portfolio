Projects

01 Parfumerie Vault

A web-based application designed to showcase and manage a collection of perfumes. My team and I focused on building a responsive front-end using HTML, CSS, and JavaScript to create an intuitive and visually appealing user interface. The project involved implementing dynamic navigation, product filtering, and interactive elements to enhance user experience, while ensuring cross-browser compatibility and mobile responsiveness.

02 Customer Management System

A small-scale web application developed to streamline customer data management. We worked on both the front-end form design and basic backend logic, using HTML forms for data input and JavaScript for validation and interactivity. This project deepened my understanding of CRUD operations, data handling, and user experience considerations in a client-server environment.

03 Python Car Inventory Program

A Python program designed to manage a car inventory, utilizing object-oriented programming principles. The program allowed for adding, removing, and updating car records, with features for sorting and searching through the inventory. This project strengthened my skills in class design, data structures (like lists and dictionaries), file handling for data persistence, and algorithmic thinking to implement efficient search and sort functionalities.
